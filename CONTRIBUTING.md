
# Contributing Guide / Beitragshinweise

---

## English

### No-Code Welcome
You don't need to know how to code to help! Have a photo tip? Found a typo? Let us know by opening an issue or editing directly.

### Ways to Contribute
- **Level 1 (Easy):** Open an "Issue" on GitHub to suggest a change or report a problem.
- **Level 2 (Medium):** Use the GitHub "Edit" button (pencil icon) to fix text directly in the browser.
- **Level 3 (Advanced):** Submit a Pull Request for new Typst templates or scripts.

### Content Guidelines
- Keep it professional and respectful.
- Follow the Feuerwehr-Ehrenkodex (Firefighter Code of Honor).
- No political content.

### Translation
If you are fluent in English or German, help us translate the `/de/` folders.

## Deutsch

### Kein Code nötig!
Du musst nicht programmieren können, um zu helfen! Hast du einen Fototipp? Einen Tippfehler gefunden? Melde dich einfach per Issue oder bearbeite den Text direkt.

### Möglichkeiten zum Mitmachen
- **Stufe 1 (Einfach):** Erstelle ein "Issue" auf GitHub, um eine Änderung vorzuschlagen oder ein Problem zu melden.
- **Stufe 2 (Mittel):** Nutze den "Edit"-Button (Stiftsymbol) auf GitHub, um Text direkt im Browser zu korrigieren.
- **Stufe 3 (Fortgeschritten):** Reiche einen Pull Request für neue Typst-Vorlagen oder Skripte ein.

### Inhaltsrichtlinien
- Bleibe professionell und respektvoll.
- Beachte den Feuerwehr-Ehrenkodex.
- Keine politischen Inhalte.

### Übersetzung
Wenn du Englisch oder Deutsch sprichst, hilf uns, die `/de/`-Ordner zu übersetzen.
